EPIC-CoGe has many features in addition to those found in JBrowse, including:
  • filter track list by name, metadata, or data type
  • manage CoGe experiments and notebooks inside EPIC-CoGe
  • export track data to local computer or to CyVerse
  • search and go to features by name
  • search tracks by min, max, or a range of values
  • search tracks for data that overlaps features
  • combine tracks by dragging and dropping
  • convert search results into marker tracks
  • save search results as new experiments in CoGe
For more infomation, see the Help menu in the JBrowse menu bar.
